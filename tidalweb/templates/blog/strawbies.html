Strawbies is a tangible programming game designed by Felix Hu, Ariel Zekelman, and Eric Uchalik. Players physical tiles to program instructions for an on-screen character called “Awbie.” Our current implementation uses an Osmo game system with an iPad for fluid tangible interaction.
